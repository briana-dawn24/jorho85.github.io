Introduction to Linear Functions

Linear equations are an important tool in science and many everyday applications. They allow scientists to describe relationships between two variables in the physical world, make predictions, calculate rates, and make conversions, among other things.

Today's Learning Objectives

  • Writing and representing linear functions
  • Determining whether a linear function is increasing, decreasing, or constant
  • Calculate the slope from two points
  • Finding the intercept of a linear function with algebra
  • Recognize when and how to re-write linear functions
  • Looking at linear relationships in a table
  • Given a graph of a line, write the equation for that line
